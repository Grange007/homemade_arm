# Motor Tools

Python interface for communicating with the motor

## Cybergear

### Example

```shell
cd Cybergear

```

## Unitree

### Build

- Add `pybind11` submodule in `./Unitree`

```shell
cd Unitree
mkdir build
cd build
cmake ..
make
```

### Example

```shell
cd Unitree
python3 example.py
```
